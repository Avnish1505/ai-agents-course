# 🏋️ Lesson 01 Exercises

## Exercise 1 — Easy 🟢
Add a new tool called `reverse_string(text)` to the SimpleAgent.
Test it with: *"Can you reverse the word 'algorithm'?"*

## Exercise 2 — Medium 🟡
Add a **step counter** to the agent that prints how many reasoning steps it took to reach the final answer.

## Exercise 3 — Medium 🟡
Add a `search_dsa_problems(topic)` tool that returns 2-3 sample LeetCode-style problem names for a given DSA topic (hardcode 5 topics).

## Exercise 4 — Hard 🔴
Modify the agent to have **persistent memory** — save `self.memory` to a JSON file after each conversation, and load it at the start of the next session.

## Exercise 5 — Challenge 🔥
Build a second agent called `CriticAgent` that reviews the first agent's final answer and rates it 1-10 for accuracy. Have both agents talk to each other!

**LeetCode Practice:** https://leetcode.com/problems/two-sum/ (warm-up!)
