# 🏋️ Lesson 02 Exercises

## Exercise 1 🟢
Count the tokens in 5 different prompts using `tiktoken`. Which type of text (code vs prose vs JSON) uses the most tokens per word?

## Exercise 2 🟡
Write 3 different system prompts for a DSA tutor agent. Test each and compare response quality. Which works best?

## Exercise 3 🟡
Implement few-shot prompting: give 2 examples of DSA explanations, then ask the agent to explain a new topic in the same format.

## Exercise 4 🔴
Build a "Chain-of-Thought" wrapper: automatically append "Let's think step by step:" to any math/logic question and compare results vs without it.

## Exercise 5 🔥
Build a prompt optimizer: take a weak prompt, ask GPT-4o to improve it, then benchmark both prompts on 5 test questions.
